## Quick Start Guide

### 1. Run ComfyUI with SageAttention
```bash
./comfy.sh run sage
```
Then open http://127.0.0.1:8188

### 2. Using RadialAttention Nodes

**Native Node (ComfyUI-RadialAttn):**
- Search for "PatchRadialAttn" in Add Node menu
- Connect to model before sampling
- Automatically accelerates video generation

**Wrapper Nodes (ComfyUI-WanVideoWrapper):**
- Search for "WanVideoSetRadialAttention"
- More advanced options and integration
- Includes TeaCache, FBCache for further speedup

### 3. Download WAN 2.2 Models

**Required Models** (place in appropriate folders):
1. **Diffusion Model** (8.4GB FP8) → `ComfyUI/models/diffusion_models/`
   - `Wan-2.2-text-to-video-FP8.safetensors`
   - `Wan-2.2-image-to-video-FP8.safetensors`

2. **VAE** (166MB FP8) → `ComfyUI/models/vae/`
   - `Wan-2.2-VAE-FP8.safetensors`

3. **Text Encoders** → `ComfyUI/models/text_encoders/`
   - `t5xxl_fp8_e4m3fn.safetensors` (4.6GB FP8)
   - `clip_l.safetensors` (246MB)

4. **CLIP Vision** (optional) → `ComfyUI/models/clip_vision/`
   - For image-to-video conditioning

**Download from Kijai's HuggingFace:**
https://huggingface.co/Kijai/Wan2.2_comfy/tree/main

### If ComfyUI Fails to Start
```bash
# Check CUDA toolkit is available
/opt/cuda/bin/nvcc --version

# Verify SpargeAttn is installed
source .venv/bin/activate.fish
python -c "from spas_sage_attn import block_sparse_sage2_attn_cuda; print('OK')"

# Re-sync environment
./comfy.sh sync
```

### If Compilation Fails (Future Updates)
```bash
# Ensure GCC 14 is used
export CC=/usr/bin/gcc-14
export CXX=/usr/bin/g++-14
export NVCC_CCBIN=/usr/bin/g++-14

# Rebuild SpargeAttn
cd /tmp/SpargeAttn
rm -rf build/
uv pip install --no-build-isolation -e .
```
